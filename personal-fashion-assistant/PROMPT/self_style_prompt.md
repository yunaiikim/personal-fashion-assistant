# Task
请分三步完成分析：

## Step 1｜视觉共性提取（结构化输出）
从以下5个维度归纳重复出现的设计语言：
• 色彩策略：主色/辅助色/点缀色的配比与饱和度倾向
• 廓形逻辑：H/X/A/O型剪裁偏好，松紧搭配节奏
• 材质组合：面料肌理对比（如哑光×光泽/柔软×挺括）
• 搭配法则：层叠/撞色/同色系/配饰点睛等技巧复用
• 细节符号：领型/袖型/腰线处理/鞋包配饰的重复元素

## Step 2｜风格关键词定位
从专业时尚分类体系中选出**3个核心关键词**，要求：
• 优先使用国际通用术语（如：Minimalist, Normcore, Quiet Luxury）
• 避免过度细分的小众标签（除非照片有强证据）
• 每个词需对应Step 1中的具体视觉证据

## Step 3｜风格成因解读
用「设计逻辑+个人特质」双视角解释：
• 为什么这3个关键词能概括提供的照片的穿搭？（引用照片细节）
• 这种风格如何呼应用户的身形特征/肤色/气质？
• 给出1条「风格进阶」小建议（保持核心+微调变量）
